# M.O.T The Label – US Collection (Beginner-Friendly Project)

---- To use tailwind for this project use *npm run dev*

## 🛍️ About This Project

This is a **simple e-commerce website clone** inspired by the M.O.T The Label US Collection page.

I built this project to practice **basic web development skills** like:

* Structuring a webpage with HTML
* Styling with CSS
* Adding simple interactivity with JavaScript

The goal is to recreate a **clean fashion website layout** where users can view products.

---

## 🎯 What This Website Does

* Displays fashion items (clothes, sets, etc.)
* Shows product names and prices
* Has a simple navigation bar (Shop, About, Contact)
* Works on both mobile and desktop screens

---

## 🧰 Tools Used

* **HTML** → to build the structure
* **CSS** → to style the website
* **JavaScript** → for small interactions
